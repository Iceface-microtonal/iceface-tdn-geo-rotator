# TDN Geo-Rotating Arpeggiator

**Microtonal toy that geometrically rotates notes** — Iceface Toys series #3

Drag to bend and rotate sounds while generating arpeggios.  
Perfectly synced drum machine included.  
Harmony and rhythm merge in real-time under the **Relational-Tonal** framework.

### 🎯 What is TDN?
Based on **Iceface TDN theory** (microtonal interval difference **N**).  
A completely new rhythmic concept where microtonal pitch differences (N) directly affect rhythm itself.

### ✨ Features
- Geometric rotation control for unique arpeggios  
- Real-time drum synchronization  
- Drag & drop sound bending  
- Mobile-friendly (works on smartphone)  
- Pure HTML / JavaScript — no installation needed  

### 🚀 How to Use
1. Open `index.html` (or visit the GitHub Pages URL)  
2. Press **Play**  
3. Drag the mesh to rotate and bend notes  
4. Drum machine automatically locks in sync
